# Tesla Model Y Juniper Custom Wraps

Custom wrap skins for Tesla Model Y 2026 Juniper display visualization.

## Structure

```
templates/           # Official Tesla UV templates
  modely-2025-base/
  modely-2025-premium/
  modely-2025-performance/
skins/               # Custom created skins (ready to use)
examples/            # Tesla's example skins for reference
```

## How to Use

1. Copy skins from `skins/` to a USB drive in a folder called `Wraps`
2. Plug USB into your Tesla
3. Go to **Toybox → Paint Shop → Wraps tab**

## Requirements

- **Resolution:** 512x512 to 1024x1024 pixels
- **File Size:** Max 1 MB
- **Format:** PNG only
- **File Count:** Up to 10 images at a time

## USB Drive Format

Use one of: exFAT, FAT 32, MS-DOS FAT, ext3, ext4  
(NTFS not supported)

## Creating Custom Skins

Use the template from `templates/modely-2025-*/template.png` in your image editor. Fill the white panel areas with your design.
